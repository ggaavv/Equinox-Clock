.. _arduino-charcast:

char()
======

Description
-----------

Converts a value to the
`char <http://arduino.cc/en/Reference/Char>`_ data type.



Syntax
------

char(x)



Parameters
----------

x: a value of any type



Returns
-------

char



See also
--------


-  `char <http://arduino.cc/en/Reference/Char>`_
