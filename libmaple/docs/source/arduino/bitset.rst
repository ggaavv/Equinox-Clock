.. _arduino-bitset:

bitSet()
========

Description
-----------

Sets (writes a 1 to) a bit of a numeric variable.



Syntax
------

bitSet(x, n)



Parameters
----------

x: the numeric variable whose bit to set



n: which bit to set, starting at 0 for the least-significant
(rightmost) bit



Returns
-------

none
