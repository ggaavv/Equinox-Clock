.. _arduino-bytecast:

byte()
======

Description
-----------

Converts a value to the
`byte <http://arduino.cc/en/Reference/Byte>`_ data type.



Syntax
------

byte(x)



Parameters
----------

x: a value of any type



Returns
-------

byte



See also
--------


-  `byte <http://arduino.cc/en/Reference/Byte>`_
