.. _arduino-bit:

bit()
=====

Description
-----------

Computes the value of the specified bit (bit 0 is 1, bit 1 is 2,
bit 2 is 4, etc.).



Syntax
------

bit(n)



Parameters
----------

n: the bit whose value to compute



Returns
-------

the value of the bit
