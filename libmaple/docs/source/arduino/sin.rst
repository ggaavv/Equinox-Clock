.. _arduino-sin:

sin(rad)
========

Description
-----------

Calculates the sine of an angle (in radians). The result will be
between -1 and 1.



Parameters
----------

rad: the angle in radians (*float*)



Returns
-------

the sine of the angle (*double*)
