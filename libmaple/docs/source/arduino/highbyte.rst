.. _arduino-highbyte:

highByte()
==========

Description
-----------

Extracts the high-order (leftmost) byte of a word (or the second
lowest byte of a larger data type).



Syntax
------

highByte(x)



Parameters
----------

x: a value of any type



Returns
-------

byte
