.. _arduino-lowbyte:

lowByte()
=========

Description
-----------

Extracts the low-order (rightmost) byte of a variable (e.g. a
word).



Syntax
------

lowByte(x)



Parameters
----------

x: a value of any type



Returns
-------

byte
