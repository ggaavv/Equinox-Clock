.. _arduino-bitread:

bitRead()
=========

Description
-----------

Reads a bit of a number.



Syntax
------

bitRead(x, n)



Parameters
----------

x: the number from which to read



n: which bit to read, starting at 0 for the least-significant
(rightmost) bit



Returns
-------

the value of the bit (0 or 1).
